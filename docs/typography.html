About WebUI

WebUI is a little different to most other CSS frameworks due to an emphasis on flexibility and creativity. Although most frameworks do attempt to provide flexibility, they often enforce a particular approach or style, and come with too many limitations. WebUI has been build from the ground up with flexibility as the primary goal - even the components are made that way. WebUI is a system that allows you to invent new designs, and provides simple building blocks to construct much larger CSS components. Complex data entry forms can be created in a precise layout using the built-in grid system that doesn't throw any surprises. Add responsive behaviour exactly where you want it, with the flexibility to create fixed non-responsive layouts or layout sections.

Be creative. Start something amazing today!

Font sizes

WebUI does not define any base font size and just uses the browser default. The following classes can be used to set different sizes relative to the browser font size or parent element font size:

text - Sets the line height to 1.5em
text-xs - Sets the font size to 0.75 times the browser font size or the parent element font size.
text-sm - Sets the font size to 0.875 times the browser font size or the parent element font size.
text-lg - Sets the font size to 1.125 times the browser font size or the parent element font size.
text-xl - Sets the font size to 1.25 times the browser font size or the parent element font size.

In order to increase or decrease the font size for the body, container, or even a single column, you just need to add one of the above classes to it. Then, any classes you use within that element will size relative to the parent font size. However, if you use the class text-md there will be no inherited change because text-md is actually 1em, which means multiply by 1.

For example, if you use text-lg on a container and specify text-lg on any columns within that container, then the text size would actually be equivalent to 20.25px for those columns, based on a browser font size of 16px.

Adding colour

You can apply any theme colours to text content for either foreground or background, and easily add effects using color-hover-* or background-hover-* classes. But if you wanted to simply highlight a section of text you could just specify the theme color name as the class. For example, info or highlight.

For more information on using theme colours please see the Theme colours section.

Paragraph formatting

The class text can be used on any element that may contain text content - this will cause the line-height to be set at 1.5em, which will give good spacing between each line.

Blockqotes are formatted with a 0.438 rem left border using the default theme color, but you can easily style them using WebUI classes.
