About Webui

Webui offers an alternative to larger frameworks, providing a small, flexible, efficient and easy to use system to create websites quickly. The philosophy is to follow a "Simplicity is the ultimate sophistication" approach, providing simple building blocks to construct much larger CSS components without the pain. Complex data entry forms can be created in a precise layout using the built-in grid system that doesn't throw any surprises. Add responsive behaviour exactly where you want it, with the flexibility to create fixed non-responsive layouts or layout sections.

Webui allows you to create your web app without getting distracted by complexity.

Font sizes

The base font size is currently 14px, and the following classes can be used to set different sizes relative to the base font size:
text - Sets the line height to 1.5em
text-xs - Sets the font size to 0.6 times the base font size or the parent element font size.
text-sm - Sets the font size to 0.8 times the base font size or the parent element font size.
text-lg - Sets the font size to 1.3 times the base font size or the parent element font size.
text-xl - Sets the font size to 1.5 times the base font size or the parent element font size.
In order to increase or decrease the font size for the body, container, or even a single column, you just need to add one of the above classes to it. Then, any classes you use within that element will size relative to the parent font size. However, if you use the class text-md there will be no inherited change because text-md is actually 1em, which means multiply by 1.

For example, if you use text-lg on a container and specify text-lg on any columns within that container, then the text size would actually be 23.66px for those columns - that's quite large.

Font styles

There are a handful of font style classes, such as text-bold, text-italic, text-upper, text-lower, etc. A full list is available int the CSS reference section.

Adding colour

You can apply any theme colours to text content for either foreground or background, and easily add effects using color-hover-* or background-hover-* classes. But if you wanted simply highlight a section of text you could just specify the theme color name as the class. For example, info or highlight.

Paragraph formatting

The class text can be used on any element that may contain text content - this will cause the line-height to be set at 1.5em, which will give good spacing between each line.

Blockqotes are formatted with a 7 pixel left border using the default theme color by default, but you can easily style them using Webui classes.
